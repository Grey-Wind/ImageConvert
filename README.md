# DeveloperTools

# 开发者工具

## 克隆项目教程

### 克隆项目

``` sh
git clone https://github.com/QingYi-Studio/DeveloperTools.git
```

### 安装依赖

```sh
pnpm install
```

#### 编译并使用热重载开发

```sh
pnpm dev
```

#### 编译并使用生产模式构建

```sh
pnpm build
```
